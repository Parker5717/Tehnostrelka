# CASPER AR Assistant

AR-помощник для адаптации молодых сотрудников на производстве.
Хакатон **«Цифровой вызов»** от Casper AI, Нижний Новгород 2026.

> **Команда:** «Команда которая объективно лучше всех, но судьи этого не замечают» 🎯

> Молодой сотрудник → камера смартфона → подсказки и квесты от ИИ → уверенная работа в цеху.

## Что это

Web-приложение, которое открывается в браузере смартфона/планшета и помогает
новичку освоиться на заводе через геймифицированные AR-квесты:

- 🎯 **Квесты-открытия** — «найди огнетушитель», «отсканируй станок №3»
- 🦺 **Safety Check** — фронтальная камера проверяет наличие СИЗ перед сменой
- 🏆 **XP, уровни, ачивки** — карьерный рост от стажёра до специалиста
- 🤖 **Маскот-наставник** — снимает стресс первых дней через короткие диалоги

## Стек

| Слой | Технология |
|---|---|
| Backend API | FastAPI 0.115 + SQLAlchemy 2.0 |
| База | SQLite (для прода — заменить на PostgreSQL в `config.py`) |
| CV (сервер) | OpenCV + YOLOv8 (ultralytics) — *добавится на шаге 4* |
| CV (клиент) | js-aruco2 для маркеров — *шаг 3* |
| Frontend | Vanilla JS + Tailwind CDN — *шаг 3* |
| Транспорт | REST + WebSocket для стрима кадров |

## Запуск шага 1

```bash
# 1. Виртуальное окружение
cd backend
python3 -m venv .venv
source .venv/bin/activate   # Linux/macOS
# .venv\Scripts\activate    # Windows

# 2. Зависимости
pip install -r requirements.txt

# 3. Запуск dev-сервера
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

Открыть в браузере:
- http://localhost:8000/        — меta-эндпоинт
- http://localhost:8000/health  — проверка живости
- http://localhost:8000/docs    — Swagger UI

## Тесты

```bash
cd backend
pytest tests/ -v
```

Шаг 1 проверяет: эндпоинты /, /health, /openapi.json и наличие всех 6 таблиц
в БД после lifespan-инициализации.

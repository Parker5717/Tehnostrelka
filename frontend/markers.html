📋 Как использовать маркеры

  1. Распечатай маркеры и прикрепи к объектам
  2. Маркер ID 0 → входная зона / дверь
  3. Маркер ID 1 → кнопка аварийного стопа
  4. Маркер ID 2 → рабочий станок
  5. Маркер ID 3 → зона хранения инструмента
  6. Войди в приложение → начни квест → наведи камеру на маркер
  7. Появится кнопка «✅ Засчитать» — нажми!
